# Weston Inc. Sales Dashboard – Excel

## Project Overview
This project is an interactive sales dashboard built in Microsoft Excel to analyze the sales performance and business metrics of Weston Inc. across regions, product categories, sales reps, and customer types.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

## Key Features
- Dynamic filtering using slicers
- Regional sales performance tracking
- Product category and profitability analysis
- Sales channel comparison (Online vs Retail)
- KPI monitoring (Total Sales, Profit, Quantity Sold, Discount)
- Profit breakdown by sales rep and customer type
- Interactive dashboard for at-a-glance reporting

## Dataset
The dataset contains sales transaction records including product ID, sale date, sales rep, region, sales amount, quantity sold, product category, unit cost, unit price, profit, customer type, discount, payment method, and sales channel.
